consul服务发现

一、什么是服务发现

调用方获取相应服务地址的过程就是服务发现

二、怎么实现服务发现

1.服务发现的多种方式

  • http api方式: curl http://127.0.0.1:8500/v1/health/service/配置中的name?passing=false

  • DNS方式

    • 下载dig包放到gopath的bin目录下：https://github.com/sequenceiq/docker-alpine-dig

    • dig @127.0.0.1 -p 8600 hello.service.dc1.consul. ANY

    • 参数说明：my_say_hi_105.service.dc1.consul，后边的service代表服务，固定；dc1是数据中心的名字，以及最后的consul名字

  • Consul Template方式

    • 在nginx等web服务器做反向代理和负载均衡时特别有用

    • curl -L https://releases.hashicorp.com/consul-template/0.19.5/consul-template_0.19.5_linux_amd64.tgz|tar -xzv -C /usr/local/bin

2.返回的信息有

  • 注册的Consul节点信息、
  • 服务信息
  • 服务的健康检查信息

三、http api

1.http://ip:port/v1 开始，下面的endpoints都是跟在v1后的

2.endpoints分为以下几种类别

kv - Key/Value存储

agent - Agent控制
    /v1/agent/checks : 返回本地agent注册的所有检查(包括配置文件和HTTP接口)
    /v1/agent/services : 返回本地agent注册的所有 服务
    /v1/agent/members : 返回agent在集群的gossip pool中看到的成员
    /v1/agent/self : 返回本地agent的配置和成员信息
    /v1/agent/join/<address> : 触发本地agent加入node
    /v1/agent/force-leave/<node>>: 强制删除node
    /v1/agent/check/register : 在本地agent增加一个检查项，使用PUT方法传输一个json格式的数据
    /v1/agent/check/deregister/<checkID> : 注销一个本地agent的检查项
    /v1/agent/check/pass/<checkID> : 设置一个本地检查项的状态为passing
    /v1/agent/check/warn/<checkID> : 设置一个本地检查项的状态为warning
    /v1/agent/check/fail/<checkID> : 设置一个本地检查项的状态为critical
    /v1/agent/service/register : 在本地agent增加一个新的服务项，使用PUT方法传输一个json格式的数据
    /v1/agent/service/deregister/<serviceID> : 注销一个本地agent的服务项

catalog - 管理nodes和services
    /v1/catalog/register : Registers a new node, service, or check
    /v1/catalog/deregister : Deregisters a node, service, or check
    /v1/catalog/datacenters : Lists known datacenters
    /v1/catalog/nodes : Lists nodes in a given DC
    /v1/catalog/services : Lists services in a given DC
    /v1/catalog/service/<service> : Lists the nodes in a given service
    /v1/catalog/node/<node> : Lists the services provided by a node
health - 管理健康监测
    /v1/healt/node/<node>: 返回node所定义的检查，可用参数?dc=
    /v1/health/checks/<service>: 返回和服务相关联的检查，可用参数?dc=
    /v1/health/service/<service>: 返回给定datacenter中给定node中service
    /v1/health/state/<state>: 返回给定datacenter中指定状态的服务，state可以是"any", "unknown", "passing", "warning", "critical"
session - Session操作
    /v1/session/create: Creates a new session
    /v1/session/destroy/<session>: Destroys a given session
    /v1/session/info/<session>: Queries a given session
    /v1/session/node/<node>: Lists sessions belonging to a node
    /v1/session/list: Lists all the active sessions
acl - ACL创建和管理
    /v1/acl/create: Creates a new token with policy
    /v1/acl/update: Update the policy of a token
    /v1/acl/destroy/<id>: Destroys a given token
    /v1/acl/info/<id>: Queries the policy of a given token
    /v1/acl/clone/<id>: Creates a new token by cloning an existing token
    /v1/acl/list: Lists all the active tokens
event - 用户Events
    /v1/event/fire/<name>: 触发一个新的event，用户event需要name和其他可选的参数，使用PUT方法
    /v1/event/list: 返回agent知道的events
status - Consul系统状态
    /v1/status/leader : 返回当前集群的Raft leader
    /v1/status/peers : 返回当前集群中同事

四、http api实战

1.安装

go get github.com/hashicorp/consul/api

2.代码

package main

import (
    "google.golang.org/grpc"
    "fmt"
    "micro_service_project/grpc_protobuf_test"
    "context"

    "github.com/hashicorp/consul/api"
)

func main() {

    config := api.DefaultConfig()
    config.Address = "127.0.0.1:8500"

    var waitIndex uint64
    client_api,_ := api.NewClient(config)
    services,_,_ := client_api.Health().Service("my_say_hi","say_hi",true,&api.QueryOptions{
        WaitIndex:waitIndex,
    })

    // 更精细的选择
    //for _,service := range services {
    //    fmt.Println(service.Service.Address)
    //    fmt.Println(service.Service.Port)
    //}
    address := services[0].Service.Address
    port := services[0].Service.Port

    url := fmt.Sprintf("%s:%d",address,port)
    fmt.Println(url)
    //http.Get("http://127.0.0.1:8500/v1/health/service/my_say_hi?passing=true")

    // WithInsecure：跳过证书的验证
    //conn,err := grpc.Dial("192.168.0.104:8080",grpc.WithInsecure())
    conn,err := grpc.Dial(url,grpc.WithInsecure())

    if err != nil {
        fmt.Println("连接错误")
    }

    client := hello.NewTestServiceClient(conn)

    rep,err1 := client.SayHi(context.Background(),&hello.HiRequest{Name:"zhiliao"})
    fmt.Println(err1)
    if err1 != nil {

        fmt.Println("返回出错")
    }

    fmt.Println(rep.Ret)
}
