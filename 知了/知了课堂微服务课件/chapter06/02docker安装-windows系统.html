dadocker安装-windows系统

一、win7、win8系统安装docker

1.下载docker toolbox

  • 下载地址：http://mirrors.aliyun.com/docker-toolbox/windows/docker-toolbox/

2.使用docker toolbox安装

  • 勾选自己需要的组件
    • Docker CLI - 客户端，用来运行 docker 引擎创建镜像和容器。
    • Docker Machine - 可以让你在 Windows 的命令行中运行 docker 引擎命令。
    • Docker Compose - 用来运行 docker-compose 命令。
    • Kitematic - 这是 Docker 的 GUI 版本。
    • Docker QuickStart shell - 这是一个已经配置好Docker的命令行环境
    • Oracle VM Virtualbox - 虚拟机。

3.测试

  • 双击Docker QuickStart 启动Docker Toolbox 终端
  • 出现User Account Control ，点击yes
  • 运行docker version

二、win10系统安装docker

1.Docker 有专门的 Win10 专业版系统的安装包，需要开启 Hyper-V。

2.解决win10家庭版没有Hyper-V的问题

  • 拷贝我提供的Hyper-V.cmd文件
  • 右键以管理员身份运行
  • 点击是
  • 在最末处输入：Y，重启电脑
  • 搜索“启用或关闭Windows功能”查看有没有Hyper-V

3.开启Hyper-V

  • 搜索启用或关闭Windows功能
  • 勾选Hyper-V

4.安装 Toolbox

  • 下载地址： https://www.docker.com/get-started

  • 点击download对应系统版本

5.运行安装文件

  • 双击下载的 Docker Windows Installer.exe文件,一路next，finish即可

三、镜像加速

1.Windows 10

对于使用 Windows 10 的系统，在系统右下角托盘 Docker 图标内右键菜单选择 Settings，打开配置窗口后左侧导航菜单选择 Daemon。在 Registry-mirrors 一栏中填写加速器地址 https://registry.docker-cn.com ，之后点击 Apply 保存后 Docker 就会重启并应用配置的镜像地址了
