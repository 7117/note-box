docker安装-linux系统

一、linux虚拟机安装

1.ubutun系统，镜像地址：

链接：https://pan.baidu.com/s/1lCU44BKmkqjuKS434tItbw

提取码：i9xg

二、关闭桌面

vi /etc/default/grub

注释掉：#GRUB_CMDLINE_LINUX_DEFAULT="quiet splash"       桌面版

在下面添加：GRUB_CMDLINE_LINUX_DEFAULT="text"            无桌面版


更新配置：update-grub

重启：reboot

三、本地安装xhell

四、安装openssh及安装vim

可以使用xshell连接：
    sudo apt-get install openssh-server

安装vim：
    sudo apt-get install vim

五、ubuntu安装docker

1.更新apt-get

  • sudo apt-get update

2.安装包允许apt通过HTTPS使用仓库

  • sudo apt-get install apt-transport-https ca-certificates curl software-properties-common lrzsz -y

3.添加源

官方： sudo curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo apt-key add -
阿里源(推荐):sudo curl -fsSL https://mirrors.aliyun.com/docker-ce/linux/ubuntu/gpg | sudo apt-key add -

4.设置Docker稳定版仓库

sudo add-apt-repository "deb [arch=amd64] https://download.docker.com/linux/ubuntu $(lsb_release -cs) stable"

查看本机的系统架构：dpkg --print-architecture

5.更新apt源索引

sudo apt-get update

出错执行：sudo apt-get upgrade

6.安装最新版Docker CE（社区版）

sudo apt-get install docker-ce

7.检查Docker CE是否安装正确

docker version

8.docker服务管理命令

# 启动docker
sudo service docker start

# 停止docker
sudo service docker stop

# 重启docker
sudo service docker restart


使用systemctl restart docker也可以

9.镜像加速

vim /etc/docker/daemon.json

修改为：
{"registry-mirrors": ["http://f1361db2.m.daocloud.io"], "insecure-registries": []}

重启docker
